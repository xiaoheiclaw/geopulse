# Signal Watchlist — 冲突降级/升级观测框架

> v1.0 | 2026-03-09 | DAG v56 baseline
> 当前: conflict_deescalation=2%, ceasefire_backchannel=2%

## 使用方法

Pipeline 每轮新闻扫描时对照本文档检查。任何信号触发时：
1. 标注触发时间和来源
2. 启动 expiry 倒计时
3. 评估是否需要调整 DAG 概率

## 4. DAG 概率调整触发规则

| 触发条件 | 调整幅度 |
|----------|---------|
| 1.1 中任意 2 个信号同时出现 | conflict_deescalation: 2% → 5-8% |
| 1.1 + 1.2 各至少 1 个信号 | conflict_deescalation: → 10-15% |
| 1.3 中 IRGC 攻击归零 + 扫雷无重布 | hormuz_blockade: 97% → 85% |
| 1.3 中 3+ 个信号同时出现 | conflict_deescalation: → 20%+ |
| 2.0 中任意 P0 信号触发 | conflict_deescalation: → 1%, 对应升级节点 +15-25% |
| 假信号出现但无后续确认 | 维持当前概率，标注"false alarm" |
