# 霍尔木兹危机：投资决策操作系统 v3.0

**日期**：2026年3月10日  
**版本说明**：从物理约束开始搭建。四层框架的因果优先级（物理 > 制度 > 博弈 > 叙事）贯穿全系统。

---

## 核心逻辑

这个冲突的持续时间和油价路径，最终由三个物理问题决定：

1. **IRGC 还能封锁多久？**（弹药存量 × 消耗率 ÷ 美军压制效能）
2. **海峡多快能恢复通行？**（扫雷进度 × 水雷密度 × 护航建立时间）
3. **缓冲能覆盖多少缺口？**（管道港口上限 + SPR 释放速率 + 绕行运力）

谢林的三个诊断（承诺锁定、失控风险、聚点缺失）解释了**为什么以上三个物理过程不会被政治决策提前终止**——但它们不能改变物理过程本身的时间线。即使 Trump 明天就想停战，水雷还在水里。即使 IRGC 明天就想停火，31 个单位的协调需要时间。

所以系统从物理问题开始，在物理答案不足以判断时，才向制度层和博弈层寻求补充信号。

## 第一层：三个物理问题

### Q1. IRGC 还能封锁多久

**约束清单**：

| 约束 | 已知事实 | 不确定性 |
|------|---------|---------|
| 水雷存量 | 估计 5000+ 枚；类型覆盖接触、磁性、声学；单价低至 $1500 | 已布设数量不明；存储位置被打击了多少不明 |
| 反舰导弹（ASCM） | Noor/Ghader/Khalij-e Fars 系列；沿岸机动发射平台 | 首轮打击摧毁了多少不明；补给能力不明 |
| 快艇群 | 数百艘；蜂群战术核心 | 被击沉约 20 艘，剩余作战能力不明 |
| 柴电潜艇 | Kilo 级 + 微型潜艇；可隐蔽布雷和发射鱼雷 | 部署位置和存活状态不明 |
| 伊朗北岸地形 | 沿岸山地提供天然隐蔽；海峡北岸全是伊朗领土 | 不可改变的地理常数 |
| 海峡航道宽度 | 可通航航道约 9km（两条 3km 航道 + 3km 缓冲） | 不可改变的地理常数 |

**综合代理变量**：IRGC 每日袭击频率（次/天）

这个数字压缩了所有弹药存量和作战能力信息为一个单一可观测量。频率下降可能意味着弹药消耗、指挥链断裂或主动降级——在 W1–W4 阶段，不需要区分原因，因为不管哪种原因，效果都是封锁强度减弱。

**每日追踪**：
- CENTCOM 公布的袭击事件数
- 袭击目标类型（油轮 / 商船 / 军舰 / 基础设施）
- 袭击地理分布（海峡内 / 海峡外 / 海湾深处）

**蒙特卡洛参数映射**：
- irgcDecayMean：当前设 6 周。每周根据袭击频率变化趋势调整（频率环比下降 > 20% → 下调 1 周；频率稳定或上升 → 不调）
- disruptionRange：当前设 [0.55, 0.90]。触雷事件发生 → 上限调至 0.95；袭击频率连续 5 天 < 1 次/天 → 下限调至 0.40

### Q2. 海峡多快能恢复通行

这是整个分析中最被低估的问题。恢复通行不是一个决策，是一个**物理过程**——需要同时满足三个条件：水雷被清除（或航道被验证安全）、沿岸导弹威胁被压制到保险公司可接受的水平、护航编队建立。

**约束清单**：

| 约束 | 已知事实 | 含义 |
|------|---------|------|
| 美军扫雷能力 | 四艘 Avenger 级专用扫雷舰 2026 年 1 月刚从巴林撤走退役。替代者仅 3 艘 LCS（Santa Barbara, Canberra, Tulsa）配扫雷任务包，模块长期延迟、未经实战验证 | **当前扫雷能力处于几十年来最低点** |
| 盟军扫雷贡献 | 英国、沙特等合计约 3–6 艘 | 增量有限 |
| 扫雷增援时间 | 从美国本土调扫雷资产到海湾需数周（速度慢，需重型运输船） | 不可压缩的物理时间 |
| 水雷的不对称性 | 布雷极快极便宜（快艇/三角帆船即可）；扫雷极慢极昂贵 | 即使 IRGC 导弹/快艇被压制，水雷可能持续数月威胁通行 |
| 1988 先例 | 伊朗布设约 150 枚水雷，一枚 WWI 老式水雷就险些击沉 USS Roberts | 一枚即可重创薄壳油轮 |
| 保险恢复的前置条件 | 需要连续无事件观察期（通常 1–2 周）+ 扫雷进展确认 | 保险恢复滞后于实际安全改善 |
| 航运公司风险评估 | 保险恢复后仍需航运公司自主决定复航 | 可能进一步滞后 |

**传导链**：扫雷进展 → 航道安全验证 → 保险恢复承保 → 航运公司复航 → 通行量恢复。每一步都有自己的滞后。总滞后不是各步之和（有并行），但也不可能短于最慢的一步。

**Q1→Q2 耦合约束**：Q2 不独立于 Q1。扫雷舰在 IRGC 反舰导弹活跃的区域无法安全作业——扫雷舰本身就是 IRGC 的高价值目标。这意味着 Q2 的时钟不是从 D1 开始走的，而是要等到 Q1 先达到某个阈值（IRGC 袭击频率降至扫雷可执行的水平）才开始。实操上：当 IRGC 袭击频率 > 2 次/天时，convoyStartMean 的倒计时不启动。

**蒙特卡洛参数映射**：
- convoyStartMean：当前设 **5 周**，从 IRGC 袭击频率降至 < 2 次/天开始计算（不是从 D1 开始）。这是整个系统中最重要的单一参数判断
- 调整规则：CENTCOM 宣布增派扫雷资产 → 下调 0.5 周；宣布任何航道完成清除 → 下调 1 周；触雷事件发生 → 上调 1 周；IRGC 专门攻击扫雷舰 → 上调 2 周

**每日追踪**：
- 是否有触雷事件（这是 Q2 的最高优先级信号——一次触雷可以让保险恢复时间线推迟数周）
- CENTCOM 是否公布扫雷进展
- 是否有额外扫雷资产向海湾调动的迹象
- 是否有编组护航的 AIS 迹象（油轮重新挂旗、护航编队形成）

### Q3. 缓冲能覆盖多少缺口

**约束清单**：

| 缓冲机制 | 硬约束 | 受硬约束限制的最大值 | 启动时间 |
|----------|--------|-------------------|---------|
| 沙特东西管道 | 延布港装载瓶颈 ~2.5 mbd | 增量 1.9–3.0 mbd | 3–5 天切换 |
| ADCOP 管道 | 管道容量 1.8 mbd，已用 1.07 mbd | 增量 ~0.7 mbd | 快速 |
| 其他管道 | 各自瓶颈 | 合计 ~0.3 mbd | 不确定 |
| 管道合计 | **延布港是总瓶颈** | **3.0–4.0 mbd** | 1–2 周到稳态 |
| SPR | 物理泵送最低 13 天；美国最大 4.4 mbd / 90 天 | IEA 协调可持续 2–3 mbd | 2–3 周 |
| 好望角绕行 | 增加 10–13 天航程；吸收全球 VLCC 运力 15–20% | 首批到货 W2–W3；稳态约 1–2 mbd | 渐进增加 |
| 非 OPEC 增产 | 页岩：DUC 4–8 周，新井 3–6 月；DUC 库存处于历史低位 | W4 前 ~0.1–0.3 mbd | 慢 |
| 需求弹性 | 弹性 -0.05 至 -0.11（正常区间）；极端区间非线性 | 取决于价格涨幅 | 即时但有限 |

**关键结构**：缓冲不是同时到位的。按时序排列（以下数字各有 ±30–50% 不确定性，用于 MC 参数设定而非点预测）：

| 时间 | 累计可用缓冲 | 主要新增 |
|------|------------|---------|
| D1–D7 | ~1–2 mbd | 管道初步切换 + 商业库存消耗 |
| D7–D14 | ~2–3.5 mbd | 管道接近稳态 + 好望角首批在途 |
| D14–D21 | ~4–6 mbd | SPR 首批到市场 + 好望角到货 |
| D21–D30 | ~6–9 mbd | SPR 稳态 + 管道满负荷 + 绕行稳定 |
| D30+ | ~7–10 mbd | 稳态上限（不含弹性减需） |

**初始条件**（起始存量，非约束）：
- 冲击前全球过剩 2–3 mbd（这是与 1979/1990 相比最重要的缓冲因素）
- 美国 SPR：4.16 亿桶（较 2022 减少 37%）
- OECD 商业库存：28.45 亿桶（高于五年均值）
- 中国附近浮仓：~4000 万桶制裁原油
- 中国 SPR 近一年持续建库
- 季节性：北半球夏季驾驶季 5–6 月启动（日历约束，将增加需求压力）

**蒙特卡洛参数映射**：
- pipelineMax = 4.0 mbd（延布港瓶颈约束）
- pipelineRampWeeks = 2.5
- sprRateMean = 2.5 mbd
- sprDelayWeeks = 2.5（13 天物理 + 政治协商）
- surplusBuffer = 2.5 mbd

**每周追踪**：
- 延布港 / 富查伊拉港装载量（AIS / Kpler）→ 验证管道到达上限
- DOE 周报 SPR 释放量 → 验证缓冲启动
- 好望角绕行船只数量和到港时间 → 验证绕行缓冲

## 第三层：制度约束（W4 开始权重上升）

物理层设定了可能性边界，制度层决定**边界内的缓冲以什么速度到位**。

| 制度约束 | 时间常数 | 对缓冲到位速度的影响 |
|---------|---------|-------------------|
| SPR 行政流程 | 最快 13 天（不可压缩） | 即使 Trump 今天下令，D13 前无实物 |
| IEA 协调释放审批 | 成员国逐一批准，1–2 周 | 影响 SPR 释放的总规模 |
| 保险精算观察期 | 无事件 1–2 周后才可能恢复报价 | 保险恢复滞后于实际安全改善 |
| OPEC+ 共识机制 | 需要紧急会议 + 全体协商 | 影响海湾国家管道增产的速度和规模 |
| LNG take-or-pay 合约 | 合约期限内难以转换 | 限制 LNG 替代的灵活性 |
| 中国炼厂进口配额 | 行政审批周期 | 限制中国额外吸收能力 |
| 国会程序 | 下一个拨款窗口、听证排期 | 政治约束的制度化时间线 |

**每周追踪**（W4 起）：
- SPR 释放执行量（DOE 周报）
- 战争险费率变化趋势（Lloyd's）
- OPEC+ 是否召开紧急会议
- 美联储 FOMC（3月19日）措辞

## 第四层：博弈层（W4 起逐步增加权重，W12+ 主导）

博弈层信号在物理层和制度层信号不足以判断路径时进入决策流程。

### 谢林三个诊断（结构解读透镜）

**诊断一：双重承诺锁定**

IRGC 看到的美方信号包：持续轰炸 + 拒绝谈判 + "unconditional surrender" + 不排除地面部队 → "美方不会停"。

美方看到的伊朗信号包：去中心化攻击持续 + 莫杰塔巴上台 + 佩泽什基安被否定 + 攻击扩至迪拜阿布扎比 → "IRGC 无法停"。

**解锁信号**：不是某一方"说"要退出，而是**信号包出现不一致**。

美方不一致的具体形式：修辞仍强硬但军事行动转向护航（言行不一致）；或任命谈判代表（行动与"不谈判"立场不一致）。

伊朗方不一致的具体形式：某些区域袭击停止而其他继续（内部不一致）；或文官渠道传递降级信号且未被 IRGC 否定（权力结构不一致）。

**诊断二：失控风险内生**

IRGC 地方单位可能自行升级目标。Trump 可能随时改变打击清单。升级不需要任何一方"决定"升级。

**升级实现的信号**：IRGC 攻击目标从油轮→能源基础设施（管道、炼厂、淡水设施）。

**诊断三：聚点需要外部注入**

美伊双边无法自发找到停止点。降级最可能通过中国或其他第三方创造可被双方识别的协调安排。

**聚点注入的信号**：中国海军向海湾方向移动；中国宣布调停特使或框架；阿曼/卡塔尔宣布促成通道安排。

### 辅助模型（有限补充）

**前景理论**（压缩为一条）：Trump 端的承诺弹性高于 IRGC 端。如果解锁发生，更可能从美方端开始。所以美方的不一致信号比伊朗方的不一致信号更值得关注。

**Allison**（压缩为一条）：IRGC 去中心化可能导致区域行为分化。追踪袭击频率时不仅看总数，也看区域分布。

### 博弈层信号清单（W4 起每日检查）

| 信号 | 谢林解读 | 优先级 |
|------|---------|--------|
| 中国实质行动（海军/经济/特使） | 诊断三：聚点注入前导 | ★★★★★ |
| 美方军事优先级转向护航 | 诊断一：承诺信号包不一致 | ★★★★☆ |
| Trump 高成本行动（任命谈判代表、SPR 释放令） | 诊断一 + 前景理论补充 | ★★★★☆ |
| IRGC 袭击区域分化 | 诊断一 + Allison 补充：承诺开始碎片化 | ★★★☆☆ |
| IRGC 目标升级至能源基础设施 | 诊断二：失控风险实现 | ★★★★★（紧急） |
| 美方对谈判渠道的态度变化 | 诊断一：从 W2 基线检测偏离 | ★★★☆☆ |

## 三条路径定义与概率

| 路径 | 物理层核心条件 | 博弈层核心条件 | 先验概率 |
|------|-------------|-------------|---------|
| A: 快速降级 | IRGC 能力 W3–W5 快速衰减 **且** 扫雷取得快于预期的进展 | 承诺锁定被不一致信号或外部聚点打破 | 30% |
| B: 渐进消耗 | IRGC 能力按半衰期 ~6 周衰减；扫雷在 IRGC 袭击频率降至 < 2 次/天后按 ~5 周推进 | 承诺锁定渐进松动 | 50% |
| C: 长期对峙 | 水雷持续威胁（扫雷瓶颈）**且** IRGC 保持最低威慑能力 | 承诺锁定完整 **且** 无聚点注入 | 20% |

注意：每条路径都有**物理核心条件和博弈核心条件**。两者必须同时满足。路径 A 不能只靠博弈层解锁（Trump 宣布胜利但水雷还在水里 → 通行不会恢复）；路径 C 也不能只靠物理层维持（水雷清除很慢但 IRGC 其他能力已崩溃 → 可以护航通过）。

## 操作流程

### W1–W4：物理层主导

**每日早间（5 分钟）**：

| 步骤 | 检查什么 | 数据源 | 决策相关性 |
|------|---------|--------|-----------|
| 1 | IRGC 袭击频率 + 目标类型 + 区域分布 | CENTCOM | Q1 的直接测量 |
| 2 | 是否有触雷事件 | 航运报告 / Lloyd's | Q2 的最高优先级信号 |
| 3 | 海峡通行量 | Vortexa / Kpler | 综合结果变量——校验你的物理分析 |
| 4 | 布伦特价格 | ICE | 市场定价是否和你的物理分析一致 |
| 5 | 记录博弈层基线 | 公开新闻 | 不据此行动，为 W4 后偏离检测建立参照 |

**本阶段仓位规则**：

| 触发条件 | 物理层逻辑 | 操作 |
|----------|-----------|------|
| 触雷事件发生 | Q2 答案恶化：扫雷时间线延长 | convoyStartMean 上调 1 周；如果已有能源超配，加波动率对冲 |
| 袭击频率连续 7 天 > 3 次/天 **且** 通行量 < 3 mbd | Q1 答案：封锁比预期更强 | 能源加仓至 22% |
| 袭击频率连续 5 天 < 1 次/天 | Q1 答案：能力衰减快于预期 | irgcDecayMean 下调至 4 周；不立即减仓（Q2 水雷问题仍在）|
| IRGC 目标升级至能源基础设施 | Q1 变异 + 谢林诊断二实现 | 波动率头寸加倍，衰退对冲加至 5% |
| IRGC 专门攻击扫雷舰 | Q1→Q2 耦合恶化：扫雷时钟被重置 | convoyStartMean 上调 2 周 |

**关键判断一：为什么袭击频率下降不等于可以减仓**——因为 Q1（IRGC 能力）只是三个问题之一。即使 Q1 的答案变好了（能力衰减），Q2（通行恢复）仍然受水雷约束。导弹和快艇可以被压制，但已经布下的水雷不会因为 IRGC 能力衰减而消失。只有当 Q2 也出现积极信号（扫雷进展、护航迹象）时，才能考虑减仓。

**关键判断二：Q2 的时钟受 Q1 约束**——扫雷舰在 IRGC 反舰能力活跃的区域无法安全作业。convoyStartMean = 5 周的倒计时不是从 D1 开始的，而是从 IRGC 袭击频率降至 < 2 次/天开始的。如果袭击频率到 W4 仍 > 2 次/天，通行恢复的实际时间线是 W4 + 5 周 = W9，远晚于基准假设。这是路径 C 概率上调的物理层条件。

## 局限性

1. **Q1 的输入极不可靠。** IRGC 弹药存量是估计值，消耗率只能从袭击频率间接推断。一个我们不知道的隐蔽弹药库可以让 Q1 的答案完全改变。

2. **Q2 可能是决定性瓶颈。** 如果伊朗已大规模布雷且美军扫雷能力如公开信息所示那样薄弱，水雷可能使通行恢复延后至 Q3 缓冲到位之后很久。这种情况下，不是油价回不来，而是回来的时间线远超大多数分析师预期。

3. **谢林诊断依赖接收方信息。** 承诺锁定分析的有效性取决于"对手如何解读信号"。我们对 IRGC 如何解读美方信号几乎没有信息。

4. **叙事层完全缺失。** 伊朗互联网封锁使我们不知道战争在伊朗民众中激活的是"反美抵抗"还是"IRGC 灾难"叙事。这可能在 W12+ 成为主导变量。

5. **本系统是决策辅助工具，不是自动交易系统。** 所有仓位决策最终由人做出。系统提供的是结构化的信号采集和解读框架，不是指令。
