# 二阶效应 (Second-Order Effects)

## 领域
通用

## 来源
系统思维 (Systems Thinking)，Donella Meadows, *Thinking in Systems*

## 核心问题
各方反应是什么？反应的反应是什么？效应会放大还是衰减？

## 分析框架
1. **直接利益方反应** — 事件直接相关方会如何应对
2. **间接利益方反应** — 盟友、竞争对手、邻国等的连锁反应
3. **市场预期** — 金融市场、大宗商品、汇率的预期调整
4. **反馈循环** — 反应是否会放大（正反馈）或抑制（负反馈）原始效应

## Prompt 注入模板
当评估传导链时，考虑：
- 谁会因此做出反应？反应是什么？
- 反应会放大还是抑制原始效应（正反馈 vs 负反馈）？
- 间接利益方（盟友、对手、市场）会如何调整？
- 是否存在被忽视的二阶、三阶传导路径？
