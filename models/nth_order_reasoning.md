# N 阶因果推演 (N-th Order Causal Reasoning)

## 领域
通用

## 来源
系统动力学 (System Dynamics), Jay Forrester; 战略推演 (Wargaming), Herman Kahn

## 核心问题
从任何触发事件出发，能否沿因果链推到第4阶以上？每一阶的传导机制是什么？

## 分析框架

### 强制推演规则
对于**每一个新事件或概率变化**，必须完成以下推演：

1. **第1阶 — 直接后果** — 事件直接导致什么？（物理因果）
2. **第2阶 — 市场/政策反应** — 市场参与者和政策制定者如何反应？
3. **第3阶 — 反应的反应** — 第2阶的反应会引发什么新的变化？
4. **第4阶 — 结构性重定价** — 如果前3阶持续，哪些资产/制度/关系会被永久改变？

### 传导链模板
```
触发 → [物理因果] → 1阶后果
  → [市场反应/政策反应] → 2阶后果
    → [反应的反应/二阶博弈] → 3阶后果
      → [结构性变化/均衡移动] → 4阶后果
```

### 传导链质量检查
- 每条链的每一阶都必须有**明确的传导机制**（不是"导致"，而是"通过X机制导致"）
- 必须标注每一阶的**时间尺度**（天/周/月/季度）
- 必须识别**放大器和抑制器**（正/负反馈）
- 如果链在2阶就断了，问自己：第3阶真的没有吗？还是我没想到？

### 常见的被忽略传导
- 原材料 → 中间品 → 终端产品 → 消费者 → 政治反馈
- 金融价格 → 企业决策 → 就业 → 社会稳定
- 政策反应 → 市场预期 → 资本流动 → 汇率 → 进口成本
- 军事行动 → 难民/移民 → 目的国政治
- 能源价格 → 化肥 → 粮食 → 社会动荡 → 政权稳定

## Prompt 注入模板
对每个分析点，强制执行4阶推演：
- 第1阶：这个事件的直接物理后果是什么？
- 第2阶：利益相关方（市场、政府、企业）会如何反应？反应的时间尺度？
- 第3阶：这些反应会引发什么新的变化？是否有反馈回路？
- 第4阶：如果持续3-6个月，哪些结构会永久改变？
- 检查：是否存在跨领域传导链（军事→能源→化工→农业→社会）？
- 检查：是否存在反馈回路（A→B→C→A加强/抑制）？
